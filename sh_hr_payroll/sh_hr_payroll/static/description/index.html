HR Payroll - Community Edition

This module helps to manage the payroll of your organization. You can manage employee contracts with a salary structure. You can create an employee payslip and compute employee salary with salary structures & salary rules. You can generate all payslips using payslip batches.

Features

  • You can generate employee payslips.
  • You can manage salary rules and salary structure.
  • Using payslip batches you can generate multiple employee payslips at once.
  • You can print a payslip PDF report.
